Stream<QuerySnapshot> scheduleSnapshots (String id)

Generate users schedules from database

Implementation 

static Stream<QuerySnapshot> scheduleSnapshots(String id) {
  return Firestore.instance
      .collection('users')
      .document(id)
      .collection('schedule')
      .snapshots();
}
